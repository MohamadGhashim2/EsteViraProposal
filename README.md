## التصدير إلى PDF

استخدم:

Browser → Print → Save as PDF

إعدادات الطباعة المقترحة:

- Paper size: A4
- Margins: None or Default depending on browser
- Background graphics: Enabled
- Headers and footers: Off
- Scale: 100% or 95% if needed
